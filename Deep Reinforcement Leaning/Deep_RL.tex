\chapter{Example Chapter Title}

This is the introductory running text for the chapter.
Chapters always begin on a new page.
Use this space to frame the chapter's goal at a high level
before descending into sections.

\section{First Main Concept}

Lead every section with prose.
Introduce the idea, its motivation, and any notation before
defining it formally in a box.
Inline math flows naturally: a model parameterized by
$\params \in \R^d$ maps an input $\xvec$ to a prediction via
$f_{\params}(\xvec)$.
Display math is used when an expression needs space:
\[
  \loss(\params) \eqdef -\frac{1}{N}\sum_{i=1}^{N}
    \log p_{\params}(y_i \given \xvec_i).
\]

\begin{defbox}[title={Cross-Entropy Loss}]
For a model with parameters $\params$ trained on
$\dataset = \{(\xvec_i, y_i)\}_{i=1}^{N}$,
the \emph{cross-entropy loss} is:
\[
  \loss(\params) \eqdef -\frac{1}{N}\sum_{i=1}^{N}
    \log p_{\params}(y_i \given \xvec_i).
\]
\end{defbox}

After a definition box, always continue with prose.
Here you would explain what this loss measures, why
minimizing it corresponds to maximum likelihood estimation,
and how it connects to what comes next.

\subsection{A Deeper Dive}

\begin{thmbox}[title={Bayes' Theorem}]
\[
  p(\params \given \dataset)
  = \frac{p(\dataset \given \params)\;p(\params)}{p(\dataset)}
  \;\propto\; p(\dataset \given \params)\;p(\params).
\]
The posterior is proportional to the likelihood times the prior.
\end{thmbox}

Note how the theorem box below does not immediately follow a
section heading — there is always at least a sentence of prose
first to give it context.

\section{Practical Applications}

Running text sets up the example and the intuition before
those boxes appear.
Consider what the KL divergence $\KL{q}{p}$ actually measures.

\begin{intbox}
Think of $\KL{q}{p}$ not as a symmetric distance
(it isn't — $\KL{q}{p} \neq \KL{p}{q}$ in general),
but as the \emph{extra bits} needed to encode samples from $q$
using a code optimized for $p$.
The asymmetry is exactly why direction matters in
variational inference and the EM algorithm.
\end{intbox}

\begin{exbox}[title={Simple Integration}]
Let $f(x) = 2x$.
Compute $\int_0^2 f(x)\dd x$:
\[
  \int_{0}^{2} 2x \dd x
  = \bigl[x^2\bigr]_{0}^{2}
  = 4 - 0 = 4.
\]
\end{exbox}

\begin{notebox}[title={Exam Trap}]
Do \emph{not} confuse $\KL{p}{q}$ with $\KL{q}{p}$.
In variational inference we minimize $\KL{q}{p_{\params}}$
(forward KL $\to$ mass-covering)
vs.\ $\KL{p_{\params}}{q}$ (reverse KL $\to$ mode-seeking).
The behavior of the approximation differs dramatically.
\end{notebox}
